# qq_music

## 介绍

该项目为本人自学完vue2后独自开发的项目，使用了vue-cil脚手架来创建，复刻了QQ音乐的客户端

## 安装

- 下载vscode开发工具
- 下载nodejs
- 安装依赖

```
npm install
```

## 运行

```
npm run serve
```

## 功能
- 音乐播放
- 音乐搜索
- 调整音量
- 登录注册
- 修改密码
- 上传头像
- 更换背景颜色
